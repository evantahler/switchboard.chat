Administrator Features

Message Stream

 

See all of your team's messages aggregated in one place! Everyone on the team can see everything that happens... as it happens, in real time!

Threaded Messages

 

Drill into the conversation with each person. Keep a record of all of the messages with each employee.

Missed-Message Notifications

 

Optionally configure notifications so that you'll never miss a message. You can be notified via SMS or email if too much time has elapsed since our system revived a message. Set up different notification rules for each team member to create a simple priority tree.
